Circuitos y plantas

Para este modelo del circuito eléctrico se implementaron diferentes tipos de sensores, los cuales son un sensor de humedad y tierra (DHT11), un sensor para medir la humedad terrestre (Monsieur sensor) y una foto resistencia para calcular si la planta está expuesto al sol o permanece en sombra. Se conectaron de manera apropiada a nuestro NodeMCU, y junto a nuestro programa hemos producido esta pagina web para monitorear el crecimiento de nuestras plantas para cunado estén listas para la cosecha.
